# JS урок 1
## З чого складається код

- ключові слова 
- ідентифікатори
- оператори 
- роздільники 
- літерали (конкретні значення)
- коментарі

### Приклад ключового слова 
```javascript 
const
// ключове слово для оголошення константи (змінна значення якої не можна замінити)
// інші: let,var
```
### Приклад ідентифікаторів 
```javascript 
day
// ідентифікатори це назви для змінних, функцій та їх паараметрів
// програміст їх вигадує сам (можуть містити літери маленькі і великі, цифри, $, _)
```
### Приклад операторів 
```javascript 
=
// різні математичні, логічні та оператори порівняння.
// також оператори можуть бути у вигляді ключових слів
```
### Приклад роздільників 
```javascript 
()
// для груповання частин коду (круглі дужки, квадратні дужки, фігурні дужки)
```
### Приклад літералів 
```javascript 
20
// це чісловий літерал (значення числового типу)
// типи бувають: числовими, рядковими, булевськими, символьними, пустими, об'єктними та інші
```
### Приклад коментарів 
```javascript 
// day.value=20
// це однорядковий коментар. коментарі використовуються для блокування частини коду
/*це
багаторядковий
коментар*/
```
## Вбудовані об'єкти

`Math` це об'єкт з математичними константами та методами.

`Date` це об'єкт який надає поточну дату і час.

`Number` це функція-конструктор яка намагається перетворювати різні дані на число. 

## Оператори 
**оператор =** - оператор який дозволяє записати в зміну значення. Після цього оператора може бути вираз, в такому разі вираз спочатку обчислюється а потім записується в зміну

**оператор ===** - оператор порівняння двох значень. Його результатом є значення true/false

## DOM - document object model
**document** - об'єкт який відповідає за всю сторінку 
**getElementById("github")** - пошук елемента на сторінці у якого є ідентифікатор github

**addEventListener()** - спеціальний метод для додавання до елементів сторінки прослуховування (контроль різних дій користувача з елементом на сторінці)
- "click" це назва події користувача
- другий параметр це функція яка виконуватиме певну дію якщо подія сталась
- параметром функції може бути об'єкт події що сталась (зазвичай називають event)
- preventDefault() це метод об'єкта події який блокує звичайну поведінку елемента (наприклад забороняє посиланню переходити на сторінку)

**confirm()** - спеціальний метод браузера який виводить поверх сторінки інформаційне вікно з повідомленням і кнопками ok/cancel (результат натискання на кнопки в цьому вікні буде true/false)

**boolean-true/false** - два протилежних значення бульовского типу данних в js 

**if/else** - конструкція розгалуження в коді (дозволяє написати два або більше набори різних дій виконається з яких лише один). За те якій набір дій виконається відповідає умова в круглих дужках

## BOM - browser object model 
**location** - це об'єкт який відповідає за адресний рядок браузера. 

**location.href** - звернення до значення записаного в адресний рядок. Це може бути локальна або глобальна адреса сайту. Використання в коді цієї властивості призводить до миттєвого переходу на вказану адресу
